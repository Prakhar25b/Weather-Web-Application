# Weather App 🌦️

A sleek, responsive weather application featuring glassmorphism design and dynamic city-specific background images.

## Features
- **Real-time Weather:** Fetches data from OpenWeather API.
- **Dynamic Backgrounds:** Automatically updates the background image based on the searched city using the Unsplash API.
- **Glassmorphism UI:** Modern, frosted-glass effect for a premium feel.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## Tech Stack
- HTML5 & CSS3 (Custom properties and Flexbox)
- JavaScript (ES6+ Async/Await)
- [OpenWeather API](https://openweathermap.org/api)
- [Unsplash API](https://unsplash.com/developers)
- [Font Awesome](https://fontawesome.com/) Icons
